## Design Context

### Users
SyncWire users are team members and managers who need to extract, track, and complete action items from meeting transcripts. Their context is often an "automation command center" where they need to see clear progress (transcript to tasks) and secure follow-through. They are focused on operational execution, not casual communication.

### Brand Personality
**Calm, precise, dependable.**
The interface should evoke confidence, efficiency, and controlled urgency. The dark, high-contrast UI and "execution" language reinforce a serious, operational feel.

### Aesthetic Direction
**Linear-style ops dashboard with Vercel-like polish.**
- **Visual Tone:** Minimal, dark-first, structured, and performance-oriented. Rounded card surfaces.
- **Theme:** Strictly dark mode for now (slate backgrounds, compact controls, task-management clarity).
- **Core Colors:** Slate (backgrounds/surfaces), Indigo (main accent), Emerald (success states), Amber (extension/request states).
- **Anti-references:** Explicitly NOT playful, consumer-social, cluttered, neon-heavy, or using loud primary reds (except for actual errors).

### Design Principles
1. ** Operational Clarity Over Flourish:** The UI must prioritize task-management clarity and data visibility over decorative elements. Keep controls compact and structured.
2. ** Dark, High-Contrast Precision:** Lean into the dark-first aesthetic (`slate-950`/`slate-50`) to reinforce the "command center" feel. Use `indigo` deliberately to guide attention.
3. ** Confident State Communication:** Use clear, consistent color coding (`emerald` for success, `amber` for pending/extensions, `red` strictly for errors) so users instantly understand task states.
4. ** Accessible By Default:** Prioritize WCAG AA contrast, ensure keyboard navigation works flawlessly (visible focus rings), and support reduced-motion where animations are used.
5. ** Vercel-Like Polish:** Maintain a high bar for finish—smooth, purposeful micro-interactions, clean rounded corners, and minimal but elegant typography (Geist font family).
